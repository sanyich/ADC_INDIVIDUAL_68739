ADC Individual Project REST API

This application provides a set of REST endpoints for user and session management.

Available Services (POST endpoints):
/rest/createaccount Create a new user account PUBLIC
/rest/login Authenticate user and generate token PUBLIC
/rest/logout Invalidate current session token AUTHENTICATED
/rest/showusers List all users ADMIN / BOFFICER
/rest/showauthsessions List all active sessions ADMIN
/rest/showuserrole Get role of a specific user ADMIN / BOFFICER
/rest/changeuserrole Change role of a user ADMIN
/rest/deleteaccount Delete a user account ADMIN
/rest/changeuserpwd Change own password AUTHENTICATED (SELF)
/rest/modaccount Modify account data (phone/address) ROLE-BASED
